Mkdocs

下载

sudo pip3 install mkdocs
Mkdocs - 检查

Cd Desktop
Mkdocs new mysite

Cd mysite
Mkdocs serve

mkdocs build - 生成

代码插件

CodeHilite
Add the following lines to mkdocs.yml: 

markdown_extensions:
- codehilite

数学公式-Mathjax

头部插入代码： 

<script type="text/javascript" async
src="https://cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.7/MathJax.js?config=TeX-MML-AM_CHTML">
</script>

尾部插入代码： 

<script type="text/x-mathjax-config">
MathJax.Hub.Config({
tex2jax: {inlineMath: [['$','$'], ['\\(','\\)']]}
});
</script>
<script type="text/javascript" asyncsrc="path-to-mathjax/MathJax.js?config=TeX-AMS_CHTML"></script>
