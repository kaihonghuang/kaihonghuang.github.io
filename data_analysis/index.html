2. 数据分析基础

2.1 数据分析的目的，流程和案例

数据分析岗位需要的技能

a. SQL语言与常见的数据库操作
b. 熟练运用R, Python, SAS等任意一种语言
c. 数据分析与可视化技能
d. 获取外部数据的能力，比如爬虫技能
e. 掌握常见的数据挖掘算法，如分类，回归，聚类等

数据分析流程

a. (企业)：问题确定-问题拆分-指标确定-数据收集-数据清洗-数据分析-趋势预测-报告方案
b. (Kaggle)：数据获取，数据存储提取，数据预处理，建模分析，数据可视化/报表

案例

EG.京东商城上戴森吹风机夜色属性与用户信息的关系 1. 数据获取——提取戴森吹风机用户消费记录，以及性别，年龄，省份等信息 2. 数据提取——提取戴森吹风机颜色属性以及用户性别，年龄，省份等星系，进行报表 3. 建模分析——颜色属性与性别的关联分析，颜色属性与年龄的关联分析，颜色属性与xx的分析 4. 数据可视化——颜色属性与性别的分布直方图，颜色属性与年龄的箱线图 5. 分析结论——xx岁-xx岁人群偏爱中国红颜色，xx岁-xx岁人群偏爱紫红色，男性偏爱酷黑色 6. 后续行动——可以按用户年龄和性别优先推送适合颜色的吹风机

3. 数据清洗和质量评估

3.1 数据清洗和质量评估

在爬取，收集，整理了相关的数据后，首先要分析数据：
1. 数据字段是什么含义的，有什么业务逻辑信息 2. 数据字段是什么类型的，其具体的量纲是什么 数值类型，类别类型，字符串类型，文本类型，图像类型，图类型等

为什么要数据清洗

  1. 并不是所有的数据都有意义
  2. 并不是所有的数据都是干净规整的
  3. 并不是所有的数据都是满足要求的(结构化数据，非结构化数据)

为什么要进行质量评估

  1. 数据字段不一定是完整的
  2. 数据字段不一定是可理解的
  3. 数据字段可能存在噪音

3.2 数据缺失值分析

缺失的原因

  1. 数据字段无法获取
  2. 数据字段传输或存储丢失
  3. 数据字段不存在或无意义

为什么数据缺失也需要进行分析

  1. 数据缺失本身就是一种信息，有数据的来源分布等信息
  2. 数据缺失比例会影响数据分布和后续的建模操作
  3. 数据缺失可能与噪音有关联

缺失值如何处理

  1. 如果字段缺失比例很高(大于99%)，可以考虑丢弃该字段
  2. 如果缺失比例不是很高，则可以考虑填充缺失值
    a. 使用特定值进行填充
    b. 使用均值进行填充
    c. 使用众数进行填充
    d. 使用插值法进行填充
    e. 使用模型预测值填充
  3. 如果是类别字段需要填充缺失值，可以将缺失值单独编码为一类
  4. 如果是数值字段需要填充缺失值，可以将缺失值编码为特殊的数值 附.树模型可以不管，神经网络和线性回归等需要填充

3.3 数据离群点分析

数据远离分布的样本点，离群点不一定是噪音，但一定是分布非常极端的样本

离群点的来源

  1. 数据采集噪音
  2. 数据来源差异

离群点检测用途

  1. 异常流量检测
  2. 异常用户检测
  3. 异常行为检测

如何处理离群点

  1. 剔除离群样本
  2. 对离群样本进行截断

离群点可以从统计分位点和可视化两种方法识别

3.4 数据关联性分析

1. 通过绘图直观可视化

a. 散点图
b. 气泡图
c. 箱线图

2. 通过计算相关性

a. pearson系数； From scipy.stats import pearsonr
b. Spearman系数
c. Kendall系数

3. 数据关联分析有什么作用，能得到什么结论

  1. 字段关联分析能得到字段与字段之间的相关性
    a. 如果字段A与字段B之间的相关性为1，则字段A与字段B重复，可以选择剔除其中一个
    b. 如果字段A与字段B之间相关性较强，则说明此两个字段分布存在逻辑关系
  2. 字段关联分析能得到字段与标签的相关性，直接说明字段的重要性
  3. 字段关联分析能够帮助进行特征筛选和数据清洗

5.探索性数据分析

5.1 探索性数据分析EDA

数据分析目的从统计或可视化的方式从数据中得到一些结论，数据分析的问题分为两类：带着目的去进行数据分析和没有目的地去进行数据分析 1. 有目的：找到数据中占比最多的类别 -Problem -> Data -> Model -> Analysis -> Conclusions 2. 无目的：不知道自己会找到什么，即探索性数据分析EDA -Problem -> Data -> Analysis -> Model -> Analysis -> Model -> ... -> Conclusions

EDA分析

在数据挖掘任务中给定了数据背景和数据字段说明，如何通过EDA发掘数据规律，数据字段与标签的关系。EDA是数据挖掘必备的步骤，其中数据可视化是有效的工具之一： 1. 数据离群点分布(散点图，分位图) 2. 一维数据分析(直方图，统计图) 3. 二维数据分析(散点图，折线图，箱线图，小提琴图) 4. 地图数据分析 5. 关联分布
